🎨 GRACULA ICONS FOLDER
═══════════════════════════════════════════════════════════════

This folder needs 3 icon files for the Chrome extension to work:

Required Files:
--------------
✓ icon16.png   (16x16 pixels)
✓ icon48.png   (48x48 pixels)
✓ icon128.png  (128x128 pixels)


How to Generate Icons:
----------------------

METHOD 1: Use the Icon Generator (Recommended)
1. Open "create-icons.html" (in parent folder) in your browser
2. Click "Download All Icons" button
3. Move the 3 downloaded PNG files into THIS folder


METHOD 2: Create Your Own Icons
1. Create 3 PNG images with the sizes above
2. Use the vampire emoji 🧛 or your own design
3. Use purple gradient background (#667eea to #764ba2)
4. Save them as icon16.png, icon48.png, icon128.png
5. Place them in THIS folder


After Adding Icons:
-------------------
Your folder structure should look like:

gracula-extension/
├── icons/
│   ├── icon16.png   ✓
│   ├── icon48.png   ✓
│   └── icon128.png  ✓
├── manifest.json
├── config.js
├── content.js
├── background.js
├── styles.css
├── popup.html
├── popup.js
└── (other files)


Then you can load the extension in Chrome!
