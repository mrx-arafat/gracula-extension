🐛 TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════════

If something doesn't work:

BUTTON NOT APPEARING?
[ ] Refresh the page (F5)
[ ] Click on the input field
[ ] Check extension is enabled (chrome://extensions/)
[ ] Open console (F12) and look for errors

API ERRORS?
[ ] Normal for free tier (rate limits)
[ ] Add your own API key (see optional step above)
[ ] Extension will use fallback mock replies

EXTENSION NOT LOADING?
[ ] Check all files are present
[ ] Verify icons folder exists with 3 PNG files
[ ] Reload extension (chrome://extensions/ → reload icon)
[ ] Check for errors in extension details

MODAL NOT OPENING?
[ ] Check browser console (F12)
[ ] Look for JavaScript errors
[ ] Verify content.js is loaded
[ ] Try different messaging platform
