═══════════════════════════════════════════════════════════════════════
WHAT EACH FILE DOES
═══════════════════════════════════════════════════════════════════════

manifest.json
-------------
• Tells Chrome this is an extension
• Defines permissions (activeTab, storage, scripting)
• Lists which websites the extension works on
• Points to content script, background script, and popup
• Specifies icon files


config.js
---------
• Stores 11 tone definitions (Default, Angry, Chill, etc.)
• Contains AI prompts for each tone
• Has platform-specific CSS selectors for:
  - WhatsApp Web
  - Instagram DM
  - Facebook Messenger
  - LinkedIn Messages
  - Twitter/X DMs
  - Discord
  - Slack
  - Gmail
  - Telegram Web
• API configuration (Hugging Face endpoint, models)


content.js
----------
• Main brain of the extension
• Runs on every messaging platform page
• Detects which platform you're on
• Finds the message input field
• Creates and positions the floating 🧛 button
• Extracts recent conversation messages
• Shows modal when button is clicked
• Sends request to background script
• Displays generated replies
• Inserts reply into input field
• Handles copy to clipboard


background.js
-------------
• Runs in the background (service worker)
• Receives requests from content script
• Builds AI prompts with context and tone
• Calls Hugging Face API
• Parses API responses
• Extracts 3 replies from AI output
• Falls back to mock replies if API fails
• Manages API key storage
• Handles rate limiting gracefully


styles.css
----------
• Makes everything look beautiful
• Purple gradient theme (#667eea to #764ba2)
• Floating button styling (50px circle)
• Modal dialog design (centered overlay)
• Tone selector grid (responsive)
• Reply cards with hover effects
• Loading animations
• Smooth transitions
• Mobile-responsive design


popup.html
----------
• Settings page UI
• Shows when you click extension icon
• Has form for API key input
• Model selection dropdown
• Feature list
• Usage instructions
• Save button


popup.js
--------
• Handles settings form
• Saves API key to Chrome storage
• Loads saved settings
• Shows success/error messages
• Validates input


icons/
------
• Contains 3 PNG icon files
• icon16.png - Shows in extension list
• icon48.png - Shows in extension management
• icon128.png - Shows in Chrome Web Store
• Must be generated using create-icons.html


create-icons.html
-----------------
• Standalone HTML tool
• Opens in any browser
• Uses HTML5 canvas to draw icons
• Creates purple gradient background
• Adds 🧛 emoji
• Generates all 3 sizes
• Download buttons for each

DEPENDENCIES
═══════════════════════════════════════════════════════════════════════

External Dependencies:
----------------------
✓ Hugging Face Inference API (free tier)
  - Default model: Mistral-7B-Instruct-v0.2
  - Fallback: Mock replies when API unavailable

Chrome APIs Used:
-----------------
✓ chrome.runtime.sendMessage()  - Communication between scripts
✓ chrome.storage.sync           - Save/load settings
✓ chrome.scripting              - Inject content scripts

No External Libraries:
----------------------
✓ Pure Vanilla JavaScript (no jQuery, React, etc.)
✓ Pure CSS3 (no Bootstrap, Tailwind, etc.)
✓ No npm packages
✓ No build process required

NEXT STEPS
═══════════════════════════════════════════════════════════════════════

1. Generate Icons
   → Open create-icons.html in browser
   → Download all 3 icons
   → Place in icons/ folder

2. Load Extension
   → Open chrome://extensions/
   → Enable Developer mode
   → Click "Load unpacked"
   → Select this folder (gracula-extension)

3. Test It
   → Go to web.whatsapp.com
   → Open a chat
   → Click input field
   → Look for 🧛 button
   → Click and select a tone
   → Watch magic happen!

4. Optional: Add API Key
   → Get key from huggingface.co
   → Click Gracula icon in Chrome
   → Paste key and save
